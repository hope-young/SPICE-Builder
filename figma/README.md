本目录是给 **Figma AI / Galileo AI / v0.dev / Lovable** 用的提示词集合。

> **作用**：用 AI 设计工具生成 SpiceBuilder 的 GUI 设计稿，辅助 PyQt5 实现。

## 使用方法

### 方案 A：Figma AI / Make Designs

1. 打开 Figma
2. 用 Figma AI / Make Designs 功能
3. **按顺序**粘贴 4 个提示词（先 `01_overall.md`）
4. 每生成一个屏幕，单独保存
5. 不满意就调整提示词重新生成

### 方案 B：Galileo AI（更擅长 UI 设计）

1. 打开 Galileo AI
2. 选 "Desktop Application" 模板
3. 粘贴 `01_overall.md` → 生成 4 个屏幕
4. 再分别用 `02_curve_viz.md` / `03_fitting.md` / `04_param_editor.md` 细化

### 方案 C：v0.dev / Lovable / Bolt（直接生成代码）

1. 打开 v0.dev / Lovable / Bolt
2. 在提示词最后加：`"Generate Python PyQt5 code for this design"`
3. 直接生成可运行的 PyQt5 代码

## 工具选择建议

| 工具 | 优势 | 劣势 | 推荐度 |
|---|---|---|---|
| **Figma AI** | 主流、生态好、能转 PyQt5 布局参考 | AI 设计一般 | ⭐⭐⭐ |

## 设计原则

无论用哪个工具，所有屏幕必须遵循：

1. **专业工程风格**（不是消费级 App）
2. **高信息密度**（工程师偏好）
3. **现代亮色主题**（参考 Figma / Linear / Vercel / Apple Design Resources 风格）
4. **多面板布局**（数据 + 控件 + 可视化并排）
5. **键盘友好**（工程师爱快捷键）

# SpiceBuilder CSV 曲线数据格式规范

## 通用规则

- 文件编码：**UTF-8**（如有中文表头）
- 分隔符：**逗号**（`,`）
- 数值格式：浮点数可用 `"1.23"` 或 `"1.23e-4"`，不接受分数/百分比
- 温度若无特殊标注，默认为 **25°C**
- 所有 CSV 必须包含 **表头行**（第一行）
- 同一文件内同一物理量必须用相同的列名（不允许同一列混合不同物理量）
- 文件名格式：`{curve_type}_{bias_condition}.csv`（下划线分隔，无空格）

---

## 1. IdVg 曲线（转移特性）

**文件命名**：`idvg_vds{Vds}.csv`，例：`idvg_vds5.0.csv`、`idvg_vds0.5.csv`

**列定义**：

| 列名 | 物理量 | 单位 | 说明 |
|------|--------|------|------|
| `vgs_v` | 栅-源电压 | V | 自变量，连续扫描 |
| `id_a` | 漏极电流 | A | 因变量 |
| `vds_v` | 漏-源电压 | V | 偏置条件（文件中为常数） |
| `temperature_c` | 温度 | °C | 偏置条件（文件中为常数） |

**示例**：
```csv
vgs_v,id_a,vds_v,temperature_c
0.00,2.545e-02,5.0,25
0.04,2.551e-02,5.0,25
0.08,2.563e-02,5.0,25
...
```

---

## 2. IdVd 曲线（输出特性）

**文件命名**：`idvd_vgs{Vgs}.csv`，例：`idvd_vgs5.0.csv`、`idvd_vgs10.0.csv`

**列定义**：

| 列名 | 物理量 | 单位 | 说明 |
|------|--------|------|------|
| `vds_v` | 漏-源电压 | V | 自变量，连续扫描 |
| `id_a` | 漏极电流 | A | 因变量 |
| `vgs_v` | 栅-源电压 | V | 偏置条件（文件中为常数） |
| `temperature_c` | 温度 | °C | 偏置条件（文件中为常数） |

**示例**：
```csv
vds_v,id_a,vgs_v,temperature_c
0.01,1.234e-01,5.0,25
0.02,2.456e-01,5.0,25
0.05,6.012e-01,5.0,25
...
```

---

## 3. CV 曲线（电容特性）

**文件命名**：
- 合并格式（推荐）：`cv.csv`
- 分立格式：`cv_ciss.csv`、`cv_coss.csv`、`cv_crss.csv`

### 合并格式（cv.csv）

| 列名 | 物理量 | 单位 | 说明 |
|------|--------|------|------|
| `vds_v` | 漏-源电压 | V | 自变量 |
| `ciss_pf` | Ciss | F | 输入电容 |
| `coss_pf` | Coss | F | 输出电容 |
| `crss_pf` | Crss | F | 反向传输电容 |

### 分立格式（cv_ciss.csv 等）

| 列名 | 物理量 | 单位 |
|------|--------|------|
| `vds_v` | 漏-源电压 | V |
| `ciss_pf` | 对应电容值 | F |

**示例（cv.csv）**：
```csv
vds_v,ciss_pf,coss_pf,crss_pf
0.0,2.345e-09,1.234e-10,1.023e-11
0.5,2.312e-09,1.198e-10,9.876e-12
...
```

---

## 4. Qg 曲线（栅电荷）

**文件命名**：`qg.csv`

| 列名 | 物理量 | 单位 | 说明 |
|------|--------|------|------|
| `vgs_v` | 栅-源电压 | V | 自变量 |
| `qg_nc` | 栅电荷 | nC | 因变量 |
| `vds_v` | 漏-源电压 | V | 偏置条件（固定值） |

**示例**：
```csv
vgs_v,qg_nc,vds_v
0.0,0.000e+00,10.0
0.5,8.234e-03,10.0
1.0,2.456e-02,10.0
...
```

---

## 5. 体二极管特性（Body Diode）

**文件命名**：`body_diode.csv`

| 列名 | 物理量 | 单位 | 说明 |
|------|--------|------|------|
| `vsd_v` | 源-漏电压（正向导通） | V | 自变量 |
| `is_a` | 体二极管电流 | A | 因变量 |
| `temperature_c` | 温度 | °C | 偏置条件 |
| `vgs_v` | 栅-源电压 | V | 偏置条件（固定值，通常为0或负偏） |

**示例**：
```csv
vsd_v,is_a,temperature_c,vgs_v
0.0,1.234e-10,25,0.0
0.2,5.678e-03,25,0.0
0.4,1.234e-01,25,0.0
...
```

---

## 6. 击穿特性（Breakdown）

**文件命名**：`breakdown.csv`

| 列名 | 物理量 | 单位 | 说明 |
|------|--------|------|------|
| `vds_v` | 漏-源电压 | V | 自变量 |
| `id_a` | 漏极电流 | A | 因变量（通常在 μA~mA 范围） |
| `temperature_c` | 温度 | °C | 偏置条件 |

## 文件命名汇总

| 曲线类型 | 推荐文件名 | 说明 |
|---------|-----------|------|
| Id-Vg @ Vds=5V | `idvg_vds5.0.csv` | |
| Id-Vg @ Vds=0.5V | `idvg_vds0.5.csv` | |
| Id-Vd @ Vgs=10V | `idvd_vgs10.0.csv` | |

| Ciss vs Vds | `cv_ciss.csv` | |
| Coss vs Vds | `cv_coss.csv` | |
| Crss vs Vds | `cv_crss.csv` | |

| 体二极管 | `body_diode.csv` | |
| 击穿特性 | `breakdown.csv` | |

---

## 导入优先级（GUI 加载顺序建议）

1. **IdVg 曲线**（阈值、跨导）
2. **IdVd 曲线**（饱和、输出电阻）
3. **Qg 栅电荷**（栅电荷特性）
4. **CV 曲线**（电容特性）
5. **Body Diode**（体二极管）
6. **Breakdown**（击穿电压）

同一类型多组偏置条件时，按文件名中的偏置值递增排序加载。
